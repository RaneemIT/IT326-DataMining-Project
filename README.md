# Employee-Attrition-DataMining
Employee Attrition Analysis Using Data Mining Techniques

## Project Description
This project aims to analyze employee attrition using data mining techniques.
Classification techniques are used to predict whether an employee will leave the organization, while clustering is applied to identify patterns and groups of employees with similar characteristics.

## Motivation
Employee attrition is a major challenge for organizations as it leads to increased costs and loss of expertise.
Understanding the factors that influence employee turnover can help organizations improve employee retention and decision-making.
